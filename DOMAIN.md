## 📝 注册步骤

### 使用 Namecheap（推荐）

1. **访问网站**
   - 打开 [Namecheap](https://www.namecheap.com)
   - 在首页搜索框输入域名，如 `lobsterdiary.com`

2. **检查可用性**
   - 点击 "Search"
   - 查看搜索结果，确认域名是否可用
   - 如果已被注册，系统会推荐其他后缀（如 .net, .org）

3. **加入购物车**
   - 选择域名（勾选需要的选项）
   - 点击 "Add to Cart"
   - 推荐附加服务：
     - ❌ Premium DNS（通常不需要）
     - ❌ Web Hosting（不需要，使用 Vercel）
     - ❌ Email Hosting（如需要可购买）

4. **结账**
   - 点击 "View Cart"
   - 输入优惠码（可选）：`NEWCOM`（新用户优惠）
   - 选择购买年限（1-10年）
   - 点击 "Confirm Order"

5. **创建账户**
   - 如果没有账户，需要创建一个
   - 填写个人信息
   - 完成邮箱验证

6. **支付**
   - 选择支付方式：
     - 信用卡（Visa, MasterCard, AmEx）
     - PayPal
     - 其他方式
   - 完成支付

7. **WHOIS 隐私保护**
   - 默认开启 WHOIS Guard（免费）
   - 这将隐藏您的个人信息

### 使用阿里云（中文用户推荐）

1. **访问网站**
   - 打开 [阿里云域名注册](https://wanwang.aliyun.com)

2. **搜索域名**
   - 在搜索框输入 `lobsterdiary.cn`
   - 点击 "查域名"

3. **检查可用性**
   - 查看域名状态
   - 如果显示 "可注册"，点击 "立即注册"

4. **加入购物车**
   - 选择注册年限（1-10年）
   - 添加到购物车

5. **结算**
   - 点击 "去结算"
   - 登录阿里云账号
   - 填写域名信息模板：
     - 个人用户：上传身份证
     - 企业用户：上传营业执照
   - 选择 DNS 解析（默认即可）

6. **支付**
   - 选择支付方式
   - 完成支付

7. **实名认证**
   - 域名注册后需要完成实名认证
   - 通常在 1-3 个工作日内完成

### 使用 GoDaddy

步骤与 Namecheap 类似：

1. 访问 [GoDaddy](https://www.godaddy.com)
2. 搜索域名
3. 检查可用性
4. 加入购物车
5. 结账

## 🔧 域名配置

### 配置 DNS（以 Namecheap 为例）

1. **登录 Namecheap**
2. **进入域名管理**
   - 点击 "Account" > "Domain List"
   - 找到您的域名
   - 点击 "Manage"

3. **配置 DNS**
   - 找到 "NAMESERVERS" 部分
   - 选择 "Custom DNS"
   - 添加 Vercel 的 DNS：
     ```
     c.vercel-dns.com
     ```
   - 点击保存

4. **等待 DNS 生效**
   - 通常需要 24-48 小时
   - 可以使用 `dig lobsterdiary.com` 检查

### 在 Vercel 配置自定义域名

1. **登录 Vercel**
2. **进入项目**
3. **设置域名**
   - 点击 "Settings" > "Domains"
   - 输入您的域名
   - 点击 "Add"

4. **配置 DNS**
   - 按照 Vercel 提供的说明配置 DNS
   - 可能需要添加 A 记录或 CNAME 记录

5. **验证**
   - 点击 "Verify"
   - 等待证书生成（自动 SSL）

## 🛡️ 域名保护

### WHOIS 隐私保护
- 大多数注册商默认开启
- 隐藏注册人信息，防止垃圾邮件

### 域名锁定
- 防止域名被转移
- 在域名管理设置中开启

## 📞 常见问题

### Q: 域名已被注册怎么办？
A: 可以尝试：
- 选择其他后缀（.net, .org）
- 添加连字符（lobster-diary.com）
- 联系当前持有者购买（通常较贵）

### Q: 中文域名可以吗？
A: 可以，如 `龙虾日记.cn`，但不推荐：
- 兼容性差（某些浏览器不支持）
- 输入麻烦
- 不利于 SEO

### Q: 需要备案吗？
A: 
- .com/.net/.org 等域名：如果使用国外服务器（如 Vercel），不需要备案
- .cn 域名：必须备案，且需要实名认证

### Q: 域名转移怎么操作？
A:
1. 在原注册商解锁域名
2. 获取转移码（EPP Key）
3. 在新注册商发起转移
4. 确认转移邮件
5. 等待 5-7 天完成转移
